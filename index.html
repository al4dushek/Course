Wprowadzenie

Witaj na kursie podstaw tworzenia stron internetowych! Dowiesz się tutaj:

  • HTML – struktury stron internetowych.
  • CSS – stylizacji i wyglądu.
  • JavaScript – interaktywności.

Kurs jest przeznaczony dla początkujących, bez wcześniejszej wiedzy.

HTML: Struktura strony

HTML (HyperText Markup Language) to język znaczników do tworzenia stron internetowych. Używa znaczników, na przykład:




    Moja strona


    
Witaj, świecie!

Podstawowe znaczniki: `

` dla nagłówków, `

` dla akapitów, `` dla linków.

CSS: Stylizacja

CSS (Cascading Style Sheets) odpowiada za wygląd strony. Przykład:

h1 {
    color: navy;
    font-size: 24px;
}

Podłączanie CSS: przez ``.

JavaScript: Interaktywność

JavaScript dodaje interaktywność. Przykład:

function powitanie() {
    alert("Witaj!");
}

Służy do manipulacji DOM, obsługi zdarzeń i wiele więcej.
