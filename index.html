Wprowadzenie

Witamy na kursie podstaw tworzenia stron internetowych z użyciem HTML, CSS i JavaScript:

  • HTML – struktura stron internetowych.
  • CSS – stylizacja i wygląd.
  • JavaScript – interaktywność.

Kurs jest przeznaczony dla studentów aby pomóc prawidłowo napisać test.

HTML: Struktura strony

HTML (HyperText Markup Language) to język znaczników do tworzenia stron internetowych. Używa znaczników, na przykład:

<!DOCTYPE html>
<html lang="pl">
<head>
    <title>Moja strona</title>
</head>
<body>
    <h1>Witaj, świecie!</h1>
</body>
</html>

Podstawowe znaczniki: <h1> dla nagłówków, <p> dla akapitów, <a> dla linków.

CSS: Stylizacja

CSS (Cascading Style Sheets) odpowiada za wygląd strony. Przykład:

h1 {
    color: navy;
    font-size: 24px;
}

Podłączanie CSS: przez <link rel="stylesheet" href="styles.css">.

JavaScript: Interaktywność

JavaScript dodaje interaktywność. Przykład:

function powitanie() {
    alert("Witaj!");
}

Służy do manipulacji DOM, obsługi zdarzeń i wiele więcej.
